# Chat App - SwiftUI

Um aplicativo de chat moderno desenvolvido em SwiftUI seguindo a arquitetura MVVM.

## Funcionalidades

- Listagem de conversas (Home)
- Visualização de mensagens e envio (Chat)
- Simulação de dados via Mock Service
- Suporte a grupos e mensagens individuais
- Design responsivo e limpo

## Tecnologias Utilizadas

- **SwiftUI**: Framework de interface declarativa.
- **MVVM**: Padrão de arquitetura.
- **Injeção de Dependência**: Para facilitar testes e modularidade.
- **Async/Await**: Para chamadas assíncronas simples e eficientes.

## Como Executar

1. Abra o arquivo `chat.xcodeproj` no Xcode.
2. Selecione um simulador (ex: iPhone 15).
3. Pressione `Cmd + R` para compilar e rodar.
